Mempercantik Terminal dengan Powerline dan ZSH

Rabu, 14 Juni 2017
tutoriallinux
programmer, terminal, zsh


Jika kamu merupakan pengguna sistem opeasi linux entah apapun distro yang kamu pakai, pasti kamu sudah tidak asing lagi dengan terminal atau bash. Bash di Linux menjadi kebutuhan primer (menurut saya) untuk melakukan berbagai operasi, sebut saja yang paling dasar seperti perintah sudo apt-get update untuk memperbaharui repositori dan sudo apt-get purge untuk menghapus aplikasi / program yang terinstal di sistem.

Karena saya seorang programmer, penggunaan bash tidak cukup sampai disitu saja, banyak kegiatan lainya seperti pengoperasian GIT untuk version control dari program yang saya kembangkan.

Singkat cerita, waktu kumpul sama temen - temen dinusopensourcecommunity (DOSCOM) saya lihat tampilan bash mereka beda dari biasanya dan lebih keren tentunya. Ketika saya tanya, ternyata mereka menggunakan ZSH dengan font Powerline. Kurang lebih bentuknya seperti ini.

Bagaimana ? Lebih keren dari bash atau terminal versi standar kan ? dan satu lagi yang saya suka dari ZSH ada fitur tab sebagai autocomplete perintah atau folder yang ada. Nah pengen coba ? mari ikuti langkah berikut ini untuk instalasinya.

Install ZSH

  1. Langkah pertama adalah Install ZSH melalui apt-get
  1. Jadikan ZSH sebagai default shell
  1. Logout dan login kembali dari sistem operasi kamu.
  2. Sekarang buka kembali terminal, dan ketika perintah

Jika keluar /usr/bin/zsh berarti sudah berhasil mengubah ZSH menjadi default. lanjut ke langkah selanjutnya.

Install Oh-my-zsh

oh-my-zsh yang membuat tampilan terminal menjadi lebih keren. berikut langkah instalasinya.

  1. Install oh-my-zsh melalui curl

* Jika ada pesan eror, mungkin kamu belum install curlnya. Silahkan install dengan perintah sudo apt-get install curl.

  1. Atur plugin dan theme

kalau saya pakai tema agnoster, dan ubah temanya menjadi

  1. Logout dan login kembali dari sistem dan buka kembali shellnya. Dan lihat hasilnya. Jika tampilannya masih kacau (tidak sesuai screenshot saya), berarti font nya belum terinstal. Silahkan install Powerline dengan cara.
  1. Selesai!
